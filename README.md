# 📄 Currículo em React Native

Aplicação mobile desenvolvida em React Native com Expo para apresentar um currículo profissional de forma moderna e organizada.

O projeto exibe informações pessoais, formação acadêmica, habilidades técnicas e projetos desenvolvidos, utilizando componentes nativos do React Native e estilização com StyleSheet.

---

## 🚀 Tecnologias Utilizadas

- React Native
- Expo
- JavaScript
- StyleSheet API

---

## 📱 Funcionalidades

- Exibição de dados pessoais
- Objetivo profissional
- Formação acadêmica
- Lista de habilidades técnicas
- Seção de projetos desenvolvidos
- Layout responsivo com ScrollView

---

## ▶️ Como Executar o Projeto

### 1. Clone o repositório

```bash
git clone git@github.com:Caio-Paschoal97/Curriculo-em-react-native.git
```

### 2. Entre na pasta do projeto

```bash
cd Curriculo-em-react-native
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Execute o projeto

```bash
npx expo start
```
